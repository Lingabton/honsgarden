# Hönsguiden — Strategi & färdplan

## Vision
Hönsguiden ska vara Sveriges go-to-sajt för hönsutrustning. Inte störst — bäst. En sajt folk bokmärker och rekommenderar vidare.

---

## Fas 1: Grundläggning (nu → juni 2026)
**Mål: 500 sidvisningar/vecka, Google-indexering av alla sidor**

### Trafik
- [ ] Request indexing i Search Console (alla 14 sidor)
- [ ] Posta i 3-5 svenska Facebook-grupper (Höns i trädgården, Hönsforum Sverige, Självhushåll)
- [ ] Svara på hönsrelaterade frågor på Reddit (r/BackYardChickens), Familjeliv, Odla.nu — länka till relevanta guider
- [ ] Skapa Google Business Profile för Hönsguiden (gratis, ger legitimitet)

### Innehåll — nya sidor som fångar sökningar
- [ ] `/guide/honsfoder.html` — "Vad äter höns?" (hög sökvolym, ingen bra svensk sida finns)
- [ ] `/guide/honsgard-bygga.html` — "Bygga hönsgård/hönsnät" (nämns i guider men ingen dedikerad sida)
- [ ] `/guide/elstangsel-hons.html` — "Elstängsel för höns" (dyra produkter = bra affiliate)
- [ ] `/guide/index.html` — Guider-hub (fixar broken breadcrumbs + ger nav-länk)
- [ ] Säsongsinnehåll: "Höns på sommaren" (april-maj är peak för nya hönsägare)

### Monetarisering
- [ ] Hitta Amazon ASINs för VEVOR, Omlet, Kerbl-produkter (3 sidor utan affiliate)
- [ ] Koppla nyhetsbrev till Buttondown/Substack (gratis, enkel setup)
- [ ] Lägg till affiliate-länkar i guider som nämner produkter utan köplänk

### Teknik
- [ ] Cloudflare security headers (Transform Rules)
- [ ] Fixa kvarvarande 14 kategori-köplänkar → riktiga produktsidor

## Fas 2: Tillväxt (juli → december 2026)
**Mål: 2 000 sidvisningar/vecka, 50+ affiliate-klick/vecka**

### Trafik
- [ ] 2-3 backlinks/månad (gästinlägg på svenska trädgårdsbloggar, Odla.nu, Byggahus.se)
- [ ] Starta Instagram @honsguiden (egna bilder finns redan, 2-3 inlägg/vecka)
- [ ] Bygg en enkel e-postlista med "Nybörjarkurs: 7 dagar till din första flock" (autoresponder)
- [ ] Pinterest — varje guide + jämförelse som pin (hönsägare är aktiva på Pinterest)

### Innehåll
- [ ] En ny guide/jämförelse per månad
- [ ] Produktrecensioner (som Run Chicken T50-sidan) för populära produkter
- [ ] Säsongsuppdateringar av alla priser (kvartalsvis)
- [ ] "Frågor & svar"-sida med alla FAQ samlat (SEO-magnet)
- [ ] Video-reviews på YouTube (länka från sajten)

### Monetarisering
- [ ] Ansök till svenska affiliate-nätverk (Adtraction, Tradedoubler) för Granngården, Bole.se etc.
- [ ] Testa "Rekommenderat startpaket"-sida med bundlade produkter
- [ ] Evaluera display ads (Ezoic/Mediavine kräver 10k+ sessions/mån)

---

## Fas 3: Etablering (2027)
**Mål: 10 000 sidvisningar/vecka, intäkter som täcker domän + tid**

### Trafik
- [ ] Google Discover-optimering (hero-bilder, E-E-A-T)
- [ ] Samarbeten med svenska hönsuppfödare/influencers
- [ ] Lokal SEO: "hönsutrustning [stad]" om relevant

### Innehåll
- [ ] Expandera till ägg (förvaring, försäljning, recept)
- [ ] Expandera till andra smådjur (ankor, kaniner) om det passar
- [ ] Prishistorik-funktion (scrapa priser, visa trender)
- [ ] Interaktivt "Vilken hönsras passar mig?"-quiz

### Monetarisering
- [ ] Display ads (Mediavine vid 50k sessions/mån)
- [ ] Sponsrade jämförelser (tydligt märkta)
- [ ] E-postlistan: veckomail med affiliate-rekommendationer
- [ ] Eventuellt: egen webshop med kurerade tillbehör

## Nyckelprinciper

1. **Kvalitet > kvantitet.** En bra guide per månad slår fem dåliga.
2. **Jag-rösten.** Sajten är personlig. Behåll det.
3. **Svenska först.** Priser i kronor, svenska butiker, svenska vintrar.
4. **Ärlig affiliate.** Dålig produkt = dålig produkt, oavsett provision.
5. **Mätbart.** Kolla analytics-rapporten varje måndag. Agera på data.
